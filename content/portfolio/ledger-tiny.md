## 背景

试过很多记账 App，最后都放弃了。原因是一样的：**太重了**。

打开 App、找分类、填金额、选日期……记一笔账要点七八下。久了就懒得记，数据断了，整个习惯也断了。

想要的很简单：打开终端，一行命令，完事。

## 使用方式

```bash
# 记一笔
ledger add 午饭 38

# 查本月汇总
ledger report

# 输出
# 2025年3月
# 餐饮    ¥ 842
# 交通    ¥ 156
# 其他    ¥ 230
# ─────────────
# 合计    ¥ 1228
```

## 技术选型

用 Rust 主要是想练手，顺便保证启动速度足够快（< 10ms）。数据存纯文本文件，一行一条记录，格式人类可读，随时可以用 grep 搜索。

## 复盘

最有价值的决定是「不做分类」。用户自己在描述里写关键词，汇总时用模糊匹配归组。减少了一个决策步骤，记账摩擦降低了很多。
